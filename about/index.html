关于本人&本站

个人资料

我是秋月夏茗，一名来自吉林省长春市的高一学生（说实话其实是萝莉呐），就读于东北师范大学附属中学。初中时我曾致力于信息学竞赛，升入高中后转为学习化学竞赛（虽然水平仍然不是那么高）。课余时间通常做奇奇怪怪的化学实验或者写一些毫无头绪的代码，有时也会学习乐理知识，比较喜欢 ACG 风格的纯音乐和 VOCALOID 歌曲，然而由于时间紧凑以及父母的要求，并不能看番。本人十分喜欢发呆，喜欢随随便便忘掉一些“不重要”的东西，似乎是个抖M。

网站简介

本博客最初是为辅助研究网络和前端开发而搭建的，后演变为兼用于记录日常感受、分享技术经验、讲解网络知识等，将来也可能加入化学和音乐有关内容。本博客由我一人撰稿、维护，运行费用（其实就是指域名费用）来自我的零花钱。本博客出于兴趣搭建，属非营利网站，无商业广告，目前不接受捐助，如果觉得小站还不错的话多向身边的人推广就好。小破站需要更多人关注┗( T﹏T )┛求求你了……

网络环境说明

本站由 Cloudflare 提供 CDN 服务。Cloudflare 与境内的互联情况可以简单地用一句话来解释：

Cloudflare is fast around the world except in China.

夜间访问本站时可能会出现卡顿。本人成年后会考虑更换境内 CDN。

网站框架&版权

本站采用纯静态框架，基于 HTML5和CSS3，存在少许较新的 HTML 标签和 CSS 样式，不支持老旧浏览器，推荐使用最新版谷歌或火狐浏览器访问。

本博客框架为本人原创，您可以自由修改或再发布本框架，无需额外标注。本博客文章无特殊说明时，均为原创，转载请先征得本人同意，并注明出处；对于非原创内容，转载要求请咨询原作者。

站点日常

本人看起来比较忙，一般在周末和假期更新文章，偶尔会在夜深人静时更换一下托管策略。通常情况下本站不存储 Cookie 或 WebStorage，也不设网页统计；但会每月接入 Cloudflare Web Analytics 收集必要的数据（不包括访问者IP等敏感信息）。
